How to Fix Maps in GrindrPlus (LSPatch Version)

            
LSPatch users experience maps issues in Grindr due to signature validation problems. Here's how to fix it:

            
            

                
    
                    
  1. Go to the Google Cloud Console.
    2. 
                    
  2. Select or create a new Google Cloud project (name doesn't matter).
    3. 
                    
  3. Make sure your project is selected in the top dropdown menu.
    4. 
                    
  4. Navigate to "APIs & Services" > "Credentials" in the sidebar.
    5. 
                    
  5. Click "+ Create Credentials" and choose "API key".
    6. 
                    
  6. Your new API key will be displayed. Click "Close".
    7. 
                    
  7. For GrindrPlus, do not set any restrictions on your API key.
    8. 
                    
  8. Copy your API key and paste it into GrindrPlus settings.
    9. 
                    
  9. Use the install button in GrindrPlus to complete setup.
    10. 
                

            

            
            

                
                Important: You may be prompted for credit card details by Google, even though use of the Maps API is part of their 'free tier'.
            

            
            

                
                API Activation Note: You may need to enable the "Geocoding API", "Maps JavaScript API" and "Maps SDK for Android" in your Google Cloud Console for your key to work properly with GrindrPlus.
            

            
            

                
                Privacy Note: This tool doesn't collect or save your API key. Everything is done on the client side, and the source code is available at https://github.com/R0rt1z2/maps-testing.
